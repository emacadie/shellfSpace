Operating Systems

Linux: There are too many distributions to list here. A good site with plenty of information about many of them is DistroWatch. For a list of some of the more popular ones, check out Desktop Linux.

Systems based on BSD 4.4:
• For performance on x86 chips and the largest number of apps: FreeBSD
• For anything that plugs into an electrical outlet: NetBSD
• For the most secure OS there is: OpenBSD
• Mach microkernel + BSD layer + world's best eye candy = Apple
• Open Darwin, the open source version of Apple's OS

Many Linux and BSD systems can be bought online at their respective websites, or on sites like Cheap Bytes, Linux Central and BSD Mall. I have seen Linux CDs at retailers such as CompUSA, Best Buy and Borders.

Eric Raymond wrote an online book on UNIX, which includes a chapter on its history.

Web Services and Related Technologies

Apple's .Mac
Dot GNU, an open-source/free software alternative to Microsoft's dot NET.
Mono, another open-source alternative to dot NET.

The most popular and widely used web tehnologies are based on Java, a language developed by Sun Microsystems (although it is possible to run Java applications that do not use any Sun hardware or software). Aside from Sun, IBM, Oracle and BEA Systems also make Java application servers and development tools.

Some popular open source Java server technologies are the JBoss application server and the Tomcat servlet engine. Shellf Space runs on Tomcat.
